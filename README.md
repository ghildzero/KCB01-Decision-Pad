# KCB01-Decision-Pad
Karawaci City Blessing Project 001

Initial idea by Louis Rusnadi; Developer by Edward Hiskia 

This project is for kids in the Church to decide what expressions they will do.
in default there are 4 expressions they can choose:
1. High Five
2. Hug
3. Shake hand
4. Kiss

the expressions selection will rotates every 2 seconds. LED matrix will show expressions animation and text.
When kid tap on the pad, the device will show animation and play audio depend on selected expressions.
